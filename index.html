零点计算方式说明:

输入"计算式"与"x的取值区间"

例如: f(x) = x - 2.5, x ∈ [1,10]

端点判断: f(1) = -1.5

f(2) = -0.5; f(1) * f(2) = 0.75 > 0 不存在零点

f(3) = 0.5; f(2)*(3) = -0.25 ≤ 0 区间[2,3]有零点，

递推表达式： 若 f(x)*f(x+1) ≤ 0,则说[x,x+1]区间存在零点;x ∈ Z+ ,x + 1 一定在区间内

表达式的输入约定：

四则运算： +、-、*、\

幂运算： ^

自然对数运算： log()

例如： 33040+23.22*x*log(x)-0.02122*x^2-492000/x
